## Description

File Upload & Auth API for "erp.aero" by Karen Kurginyan

## Installation

```bash
$ npm install
```

## Enviroment file

Create .env file with enviroment variables.
Example in .env.example file.

## Initial db

```bash
$ npx prisma migrate dev --name init
$ npx prisma generate
```

## Running the app in development

```bash
$ npm run dev
```
